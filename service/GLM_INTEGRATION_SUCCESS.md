### 🔄 系统工作流程

```
中文测试步骤 → 智谱AI理解 → 生成Playwright代码 → 安全执行 → 返回结果
```

**详细流程:**
1. 用户输入: "导航到示例网站 example.com"
2. 发送到智谱AI: POST https://open.bigmodel.cn/api/anthropic/v1/messages
3. 智谱AI理解: 识别为导航操作 + 提取URL
4. 生成代码: `await page.goto("https://example.com")`
5. 执行代码: Playwright执行导航
6. 返回结果: "Navigated to https://example.com"
